# Lim Fluoryynx - Portfolio Website

This is a modern, professional portfolio website showcasing my experience, projects, and skills with an enhanced design and interactive elements.

## Deployment to GitHub Pages

1. Create a new GitHub repository (or use an existing one)
2. Push this code to your repository
3. Enable GitHub Pages in your repository settings:
   - Go to Settings > Pages
   - Select source as "Deploy from a branch"
   - Choose the main branch
   - Click Save

Your portfolio will be available at: `https://<your-username>.github.io/<repository-name>`

For a user page (instead of project page), use the repository naming convention: `<your-username>.github.io`

## Features

- Modern, visually appealing design with gradient backgrounds
- Responsive layout that works on all devices
- Interactive navigation with smooth scrolling
- Animated elements that appear when scrolled into view
- Professional presentation of experience, projects, and skills
- Mobile-friendly navigation with hamburger menu
- Print-friendly styles
- Accessible design with proper focus states
- Enhanced contact section with improved layout
